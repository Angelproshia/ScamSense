About ScamSense

Project overview, methodology, and technical details

Project Overview

ScamSense is an AI-powered fraud detection system that identifies suspicious UPI transactions and phishing SMS messages in real time.

Digital payment fraud in India has grown significantly with the rise of UPI. ScamSense applies Machine Learning and Natural Language Processing to help users identify potential fraud before it causes financial harm.

Technical Approach

Transaction Fraud Model
  • Algorithm: Random Forest Classifier
  • Features: Amount, transaction type, time of day, round-number flag, high-amount flag, late-night flag
  • Training: Simulated realistic transaction dataset with class balance
  • Accuracy: {{ fraud_acc }}%
SMS Phishing Model
  • Algorithm: Logistic Regression
  • Features: TF-IDF unigrams and bigrams (3000 features)
  • Training: 100 labeled SMS messages (50 phishing, 50 legitimate)
  • Accuracy: {{ sms_acc }}%

Known Limitations

  • The fraud model is trained on simulated data — a production system would use real anonymized transaction data from financial institutions.
  • The SMS dataset is small (100 samples). A larger labeled corpus would significantly improve robustness.
  • The system does not have access to real-time bank APIs or transaction networks.
  • Fraud detection recall is 63% — meaning some fraud cases may not be flagged. This is a known trade-off between precision and recall.
